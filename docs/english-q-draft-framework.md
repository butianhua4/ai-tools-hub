## Page Template

### What this problem means

Purpose: Match the searcher's exact question before adding context.

- Use one direct sentence.
- Avoid brand hype.
- Mention the affected tool or platform when present.

### Quick fix

Purpose: Give the fastest safe action for urgent searchers.

- Keep it above the fold.
- Use bullets or numbered steps.
- State when the fix does not apply.

### Step-by-step solution

Purpose: Convert a quick answer into a complete troubleshooting path.

- Use short steps.
- Include checks before destructive actions.
- Keep commands copyable.

### Risks and verification

Purpose: Reduce bad fixes and improve trust.

- Add rollback notes.
- Add production cautions.
- End with a verification command or visible success condition.

### Related problems

Purpose: Strengthen the Q -> Blog -> Cluster -> Q loop.

- Link one cluster hub.
- Link one deep source article.
- Link three related English q pages when they exist.

- Every English q page links to one cluster hub.
